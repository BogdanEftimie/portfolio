About me

Who I am & what I build

I'm a BSc (Honours) Computing and IT (Software) graduate from the Open University, currently building web and mobile applications. I work across the full stack from React and Next.js frontends through to custom backend logic and REST API integrations.

Most recently I've been building Gym Log, a React Native workout tracking app. It's been a valuable push into mobile development, involving local data persistence, custom UI components, and a full backup/restore system. Shipping soon on iOS and Android.

I care about code that is readable, maintainable, and actually solves the problem at hand. I'm looking for a team where I can contribute from day one and keep growing.
